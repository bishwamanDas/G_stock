# 🏡 G Stocks – A responsive investment website

This project develops a fully responsive and interactive investment website designed to showcase financial services, investment features, and company offerings. Using modern frontend technologies, it follows a complete end-to-end workflow — from structuring the layout and designing the interface to implementing interactivity, responsiveness, and user-focused features.

The project also emphasizes responsive design and interactive UI elements to enhance user experience, accessibility, and overall performance across different devices.

---

## 📊 Project Overview

The main objective is to build a responsive and interactive investment website using modern frontend technologies, focusing on delivering a clean user interface and seamless user experience.

### ✅ Key Highlights

- 📱 Responsive layout design for all screen sizes 
- 🎨 Clean and modern UI implementation
- ⚡ Interactive elements using JavaScript and jQuery
- 🎞️ Carousel and slider integration (Owl Carousel) 
- 🎥 Video popup functionality (Venobox) 
- ✨ Scroll-based animations and dynamic UI effects 

---

## 🧰 Tech Stack

- **HTML5** - structure 
- **CSS3** – styling and layout  
- **JavaScript (jQuery)** – interactivity
- **Bootstrap** – responsive framework
- **Owl Carousel** – sliders and carousels  
- **Venobox** – video popup functionality
